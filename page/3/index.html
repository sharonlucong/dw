论学习

发表于 2017-09-11

上学的时候一直把学习当成任务，自然没有感受到多大乐趣，反而是年纪越大越开始喜欢学习。从一无所知到略知一二的过程是能让我感受到巨大的成就感的。然而随着逐步深入，仿佛走向一个深不见底的大坑，目之所及是星星点点的火光，那是拿着火炬远远走在前面的人。每每此时，又觉得异常沉重，开始质疑自己的学习能力。胡适写的歌词里说过“怕什么真理无穷，进一寸有一寸的欢喜”，这已然成为我给自己的心理暗示了。大部分的时候很管用，偶尔因为自己急迫的欲望，又被抛之脑后。

论选择

发表于 2017-09-08

一般说来，人们是不大喜欢“变动”的。为什么变动要打上引号呢？因为这里说的变动是个动作，不代表状态。举个例子来说，A处在一个安逸养老的环境中（状态），B处在一个年轻有活力的环境中（状态），如果忽然让A、B交换环境（动作），恐怕都会感到不适应。但是孰好孰坏呢？这很难说。

不过在充满挑战且不断创新变化的环境里，即使懒惰，也多少会被浪潮推动着往前（即所谓追随时代的步伐），但却很容易迷失，看不清什么是自己想要的。 在死水一般的环境中，却不可避免的“不思进取”，“原地踏步”，然而心却易平静，没有“乱花渐欲迷人眼”的扰攘。

人是个贪心的动物，不想被无情的抛下，也不想疲于奔命，所以才讲究折衷。只不过一折衷，就少了很多意思，沦落为平庸的选择。

第一篇之命名篇

发表于 2017-09-07

有时候烦闷至极，是不愿意干什么“正事”的。搭个博客，一切从简，记录一下胡思乱想。

明明是一时兴起，还装模作样的来点讲究，为这博客的名字。自己想来还是缺点文采，绞尽脑汁都想不出什么惊天地泣鬼神的名号，干脆“拿来主义”借用鲁迅《呐喊》序，断章取义的补我之所需。

“假如一间铁屋子，是绝无窗户而万难破毁的，里面有许多熟睡的人们，不久都要闷死了，然而是从昏睡入死灭，并不感到就死的悲哀。现在你大嚷起来，惊起了较为清醒的几个人，使这不幸的少数者来受无可挽救的临终的苦楚，你倒以为对得起他们么？”

既然是“断章取义”，我便只用这词语来描述我的心境，这熟睡的人是我，这较为清醒的也是我。我只是在想，是否我也应该抱有希望，能冲破我思想的铁屋子。倘若有误入此博客之人偶然瞥见我这呓语，也请不要太认真。

我感到未尝经验的无聊，是自此以后的事。我当初是不知其所以然的；后来想，凡有一人的主张，得了赞和，是促其前进的，得了反对，是促其奋斗的，独有叫喊于生人中，而生人并无反应，既非赞同，也无反对，如置身毫无边际的荒原，无可措手的了，这是怎样的悲哀呵，我于是以我所感到者为寂寞。

偶然的契机，重新开始读鲁迅的文章，加之现在庸人自扰，反而更增添感触。时代不同，我或许无法拥有像他那么强烈的使命感和责任感。在如今这个和平年代，大家都乐于经营自己的小日子，各家自扫门前雪，莫管他人瓦上霜。然而这并没什么不好不是吗？但偶尔想来，还是觉得莫名无奈悲哀，明明自己也没有多高的觉悟和使命感。只是越发敬佩那些以最赤诚的心（不带有利益）为人类的进步而奋斗不止的人们。

倘若我所做的事情，或未来做的事情是有意义的，那么便有一分热，发一分光。愿自己不忘初心，不要半途而废。

我每看运动会时，常常这样想：优胜者固然可敬，但那虽然落后而仍非跑至终点不止的竞技者和见了这样的竞技者肃然不笑的看客，乃正是中国将来的脊梁

自勉。
