混乱的大脑和心

发表于 2018-02-05

世上本无事，庸人自扰之。这么看来，我真是个彻头彻尾的“庸人”。每天都有无数的烦恼困扰，做不到所谓的“轻装前行”。一种夹杂着各种情绪的紧迫感在节日前更加汹涌澎湃的袭来。

最近我一直在想，想自己希望成为一个什么样的人。找不到什么明确的答案，便寄希望于寻求一个“榜样”，无奈的是求而不得。

我也在想人与人之间的关系，除了“血缘”这种刻在骨子里的强连结，一切都是如此弱连结。加缪在《鼠疫》中说过：

“倘若我们当中哪一位偶尔想与人交交心或谈谈自己的感受，对方无论怎样回应，十有八九都会使他不快，因为他发现与他对话的人在顾左右而言他。他自己表达的，确实是他在日复一日的思虑和苦痛中凝结起来的东西，他想传达给对方的，也是长期经受等待和苦恋煎熬的景象。对方却相反，认为他那些感情都是俗套，他的痛苦俯仰皆是，他的惆怅人皆有之。”

我深以为然。

我想自己对于事物的看法应该是偏消极和悲观的，性格中的畏首畏尾和优柔寡断加剧了我对于痛苦的体验。我觉得现在的自己仿佛是溺水中挣扎，出于本能的对于求生的欲望。当然，我仍然希望在这挣扎中，有渺小的成功几率找到水中生存的技巧，而不是放弃挣扎。但有一点很有意思，放弃挣扎不一定意味着沉入水底，还有可能在水面漂浮，随波逐流。或许有的人早就看明白这件事，甘心的顺应流动。而有的人，穷尽一生，筋疲力尽却并没有什么不同。究竟哪一种好，哪一种不好，又或者这个真的有好坏之分吗？我所说所想所写的一切，都是“旁观者”视角，而在水中的我，无论如何都不可能“旁观”地看自己吧。
