又是一周闲扯时

发表于 2018-03-05

最近闲的无聊，看起了吐槽大会。似乎每过一段时间，都会想什么也不想，就想听段子找找乐。因为听李诞的吐槽，我又想起许知远和李诞的那期对话，觉得他在那个节目里真的走心了。

李诞说：“有段时间做什么都没劲，觉得回内蒙喝三块一瓶的啤酒也行。最可怕的就是这个“也行”，因为在我心里，这个也行真的行”

其实大部分的人都在过着自己心目中“也行”的生活，或许曾经内心都有过挣扎和抵抗，尝试去过“理想”的生活，但最终都放弃了。我想那种破釜沉舟式的追求“理想”和全心全意“放弃”的人都是幸福的，最可悲的莫过于我这种左右摇摆，还无法自洽的状态。我知道，这种纠结是存在在大部分人身上的。想到这一点，我既为自己不是一个人无病呻吟而聊以自慰，也为自己只是这平凡芸芸众生中的一员并无与众不同而不甘和无奈。

顾城说: “中国是个过于明白的民族，从老子说“天地不仁，以万物为刍狗”起， 他们就意识到了宇宙和他们没有情意上的联系，人不过是宇宙中的偶然现象，无限渺小，知与不知并无异样，结论早就有了，剩下的事只是处理人间的这一小段生活而已，是相互厮杀还是保持平定，有人操心将小人间安稳地保持延续下去，有人则独善其身或依从个性，当然也有人胡作非为，强中更有强中手。他们从不期待这个世界以外的事情。”

他还说“中国人差不多都可以感到一切皆幻”，我想，这也是中国人普遍没有西方或者穆斯林那种宗教信仰的原因。看，不管在说什么的时候，不管是谁说，我们总是会说xx人这种以一个群体为单位进行归类和叙述。当然，我们也毫无悬念的发现群体中的确存在共通性。那么，当我们审视自己的时候，是以怎样的态度来面对自己身上那些共通性呢？面对劣根性的时候，是以其为自己开脱吗？但是仔细想一想，集体的意识也是由个人意识组成，再由一部分人进行引导，细思极恐。

我又没有什么中心思想的胡乱扯了扯，也不知道为什么扯，扯了有什么意义，可能只是希望自己还能保持一种“思考”的习惯吧。

平等和自由

发表于 2018-03-01

神神叨叨地在这胡乱说几句关于平等和自由的话。以前我从来没有认真想过原来这两个词互为反义词。平等即不自由，自由便不平等。如果极端一点，做一个二选一的选择题，那么哪一个更重要呢？我现在明白人都是自身利益的追逐者，哪个对自己更有利，哪个就更好。只是没有一个单一的选择可以应用于任何场景。我常说，我想要自由，不想受限制，但是绝对的自由带来的可能是更可怕的碾压和限制。我也幼稚的想过，人人应该平等，但平等意味着压抑自由发展，剥削一些人的利益。所以，哪有什么纯粹的自由和平等，人类高举大旗，慷慨激昂的站在道德制高点上宣扬自己对这两种概念的仰慕和追求，说到底本质都是为了获取利益而进行剥削。

一行路人偶然路过，想要讨口水喝，你斟上一壶自己苦心酿造的名为才华的酒，倒给他尝一尝，他说：呵，这水的味道真奇怪！

年末呓语

发表于 2018-02-15

其实按照国内时间看，现在已经是新的一年了。但现在我还是借由一点小小的时间优势，发一篇处于“年末”的感言。最近一阵子过得十分慵懒，想说先等着“变动”时刻到来的那一刻。然而，目前看来，并没有什么太多进展。

说是“新的一年”马上到来，但这都已经二月中旬了，一年的第一季度都已经过半。我想随着成长，时间相对的对于自己来说变得越来越快了，这很符合那个“相对时间认知”的概念。但说来说去，我还是没法免俗的视这个时间段为一个“新的阶段的开端”。

回望“鸡年”，在年末幸运的迎来一个变动的契机。此外，我认为自己开始关注于“内心”或者说“精神”的成长。说到这个，我一直认为自己过于畏首畏尾，胆小慎微。即使连预约个医生这种事，对我来说都算是跨出“心灵舒适区”。想来我这个“恐惧症”也是够严重的。不过环境所迫，有时候还是不得不逼迫自己采取一些行动，就拿打电话预约这件事来说，打了以后发现“其实也还好”。我想我自己很明白所有的事情都是这样的“套路”：对于好的事情来说，要发生还未发生的状态最美好，对于坏的事情来说，要发生没发生最恐怖。因为凡事发生过以后，都是一种“也还好”的状态。这让我相信，刨去“生物性”，人生意味着各种体验。当然，尽管我能如此冷静的分析这些事，我仍然不认为自己在真实的生活中，能够坦然妥善的面对各种状况。

展望“狗年”，我最期盼的是自己内心的成长，希望自己的精神足够强大来面对各种不安全感所带来的躲避和妥协。我仍需努力直视自己的内心，因为即使是在这里写下这些话，我仍然在自欺欺人式的隐瞒和逃避一些事实。希望能够理清一些思路，有一些宏观上的人生规划，来帮助我克服种种不安全感。希望自己能顺利平稳的适应一些变动。🍀 Good luck to myself!

混乱的大脑和心

发表于 2018-02-05

世上本无事，庸人自扰之。这么看来，我真是个彻头彻尾的“庸人”。每天都有无数的烦恼困扰，做不到所谓的“轻装前行”。一种夹杂着各种情绪的紧迫感在节日前更加汹涌澎湃的袭来。

最近我一直在想，想自己希望成为一个什么样的人。找不到什么明确的答案，便寄希望于寻求一个“榜样”，无奈的是求而不得。

我也在想人与人之间的关系，除了“血缘”这种刻在骨子里的强连结，一切都是如此弱连结。加缪在《鼠疫》中说过：

“倘若我们当中哪一位偶尔想与人交交心或谈谈自己的感受，对方无论怎样回应，十有八九都会使他不快，因为他发现与他对话的人在顾左右而言他。他自己表达的，确实是他在日复一日的思虑和苦痛中凝结起来的东西，他想传达给对方的，也是长期经受等待和苦恋煎熬的景象。对方却相反，认为他那些感情都是俗套，他的痛苦俯仰皆是，他的惆怅人皆有之。”

我深以为然。

我想自己对于事物的看法应该是偏消极和悲观的，性格中的畏首畏尾和优柔寡断加剧了我对于痛苦的体验。我觉得现在的自己仿佛是溺水中挣扎，出于本能的对于求生的欲望。当然，我仍然希望在这挣扎中，有渺小的成功几率找到水中生存的技巧，而不是放弃挣扎。但有一点很有意思，放弃挣扎不一定意味着沉入水底，还有可能在水面漂浮，随波逐流。或许有的人早就看明白这件事，甘心的顺应流动。而有的人，穷尽一生，筋疲力尽却并没有什么不同。究竟哪一种好，哪一种不好，又或者这个真的有好坏之分吗？我所说所想所写的一切，都是“旁观者”视角，而在水中的我，无论如何都不可能“旁观”地看自己吧。

戒浮躁

发表于 2018-01-29

“乱花渐欲迷人眼，浅草才能没马蹄” 不谈这句诗本意，想联想下引申的含义：纷纷扰扰的世界各种繁杂让人无所适从，不知如何前进，但换个角度看，乱花长在浅草上，这草才刚刚盖过马蹄的高度。也就是说，在那样的世界中，真正挡住你前进的是你自己，而不是什么花花草草。

我（们）为什么这么容易被干扰？我想其中一个原因可能是有关于信仰缺失。不管是聊天故作深沉还是别的什么时候，提到或想到信仰缺失的次数并不少，但老实说我从来没有好好想过这意味什么。或许，我一直默认把“信仰”这个大概念等同于了“宗教信仰”，因为自己并无宗教信仰，所以顺势认为自己也没什么信仰。现在我再一想，说“信仰缺失”是不准确的，应该说缺乏一种具有稳定性的信仰。现代人，崇尚金钱，崇尚社会地位，崇尚享受，这些其实也未尝不可说成是信仰。但是建立于这些之上的信仰是依托于“比较”而生，因为它们的相对价值大于绝对价值。崇尚金钱，那么我们就希望“比别人更有钱”，崇尚地位，我们就希望“成为人上人”，依次类推。信仰无时无刻不在心中，那么这些“基于比较”的信仰就成为了一种负担，因为我们无时无刻不在比较。

信仰宗教的人，或者是崇尚“士大夫精神”的古人，对于这些人来说，“信仰”体现出的是一种绝对价值，不管是“爱”或是“气节”或是类似的，“有没有”比“有多少”似乎更加重要。如果所有人都有相同的“气节”或者“爱”，这并不影响“气节”和“爱”本身的价值，但是所有人都有一样的“钱”和“地位”就会使得“钱”和“地位”无意义。所以，我想只要找到自己认为具有绝对价值的“信仰”，才不容易让自己疲于奔命和迷失。

接上一篇

发表于 2018-01-26

昨天写了一些关于优秀的看法，由此我忽然联想到一些问题，关于下一代，关于传承。我想，追求优秀是一种写在基因里的特质，本质上可能是为了拓展探索人类生存的可能性。这种基因可能是隐形存在，但一定会在某些人身上表现明显的显性特征。

很多家长都在努力的培养自己的下一代，所以我们会有“精英教育”，“各种报班”的现象存在。这是一种群体现象，更多体现在受过一定教育的父母身上。一种比较表象的解释为父母由于自身社会经验，发现这种培养所带来的“技能”或“能力”是有利于获取更多的“社会资源”，便于“社会生存”的。这种“社会生存”对于这些人来说早就不是“活着”这种非常“低级”的段位，而是具有一定阶级性的。

看到现在的家长教育子女，联想到自己以及身边人的经历，不管群体意识的具像化表现是什么方式，我始终认为这种对于优秀的追求本身是正面和积极的，因为追根溯源表现到物种层面上，这就是有利于物种延续的。当然，这是出于一种理性分析和认识。从情感上来说，这种对于“优秀”功利性和目的性的追逐让我觉得恐慌。早已拥有优渥社会资源的群体要让子女持续保有这些资源，处于相对劣势地位的群体“使劲”争取和抢夺资源，这或许就是所谓的存在“阶级流动性”的可能？我想到今天看到知乎上一篇分析提到了未来“阶级固化”的必然性，那么这种“努力争取”多多少少都有些“最后的机会”的意味。什么阶级，什么优秀，这些从本质上来说还是“生物”层级的讨论。让我恐惧和可悲的是，“人”这个窗户纸轻轻一捅就破了。

新年第一篇

发表于 2018-01-25

没想到我已经好久没有更新过了，这作为2018年的第一篇的写作动机也不过是工作时开小差的灵光一现。此时此刻，我忽然想到一个关于成长和人生轨迹的问题。人应该过怎么样的一生呢？人应该成为怎么样一个人呢？我想以前的自己会毫不犹豫的说：成为一个优秀的人。上知天文，下晓地理，对万事万物可以做出独到深刻的评价，爱好丰富，且均能小有成就。有品位，有修养，有良好的审美。我想这种期待或者说愿景本身其实是非常美好和积极向上的，但是我越来越不能确定这种对优秀的渴望来自于什么，对于可能的解答我又本能的羞愧和惶恐。我想在一定程度上这种“追逐优秀”一定是有自发性的，但我也越发确定有很大程度上我是在向往着来自外界的肯定，或者更直白的说，我的内心不自觉的在对人进行分级，而我自己迫不及待的希望能成为“靠上”的那一级。这种迫不及待让我急功近利，让我静不下心来，让我无法沉淀。我相信很多人都和我一样，对所谓成功有一种天上掉馅饼式的渴望。然而，出于某种不愿流俗的清高，对于公式化的成功定义，又摆出一副众人皆醉我独醒的拒绝姿态。用一个词来形容就是“欲拒还迎”，再为其加个形容词就是“虚伪”。

那种厚积而薄发的道理不是不懂，只是我太着急了以至于一放慢脚步就会开始心慌。然而奇怪的是，对于各种打发时间的琐事我好像反而沉溺其中，并无浪费时间的愧疚，至少说在做这些事情的时候并没有什么感知。我想我现在是享受这种只用小片段时间就可完成的事，而这件事是什么其实无关紧要。我宁愿一天做非常多“碎片式”的“无营养”的事，也不愿意花非常多时间做一件“有意义”的“大事”，这大事可以是读一本书，或者是画一幅画，或者是写个side project。这种自欺欺人式的方式，我做的也不彻底，否则也不会在这里写下这些胡言乱语。

每每想到这些，我就会觉得心慌气短。其实混日子本身只是一种生活方式，对于生存生活来说，没什么好坏高下之分。我最大的问题就在于其一，我渴望一种不同于此的人生，而行动上却做着完全相反的事。其二，我希望自己能跳出这种依托于攀比之上的“成功”模式，却往往进入“迷魂阵”中找不到方向。其三，格局太小而无法做到风物长宜放眼量。

这新年里愿望之一便是能找到切实可行的解决之道。

年末

发表于 2017-12-29

2017年马上就要结束了，最应景的是结尾带来了分别，宣告了可能性极大的阶段性结束，新的一年来临，要告别，要开启新的旅程。

可能是咖啡喝的有些心慌，此时自己觉得有些无所适从。到现在了，似乎越来越能坦然的面对人生就是不断的遇见和分别，也似乎能明白了说到底每个人都是独行侠，被赋予了时间冲淡一切的云淡风轻能力。

处在2开头年纪的尾端，仿佛丢失了满头热血的莽撞，只能怅然若失的迷茫着。凡事总要左右衡量，凡事总是优柔寡断，畏畏缩缩的原地踏步。我明白的，想走出这个困局需要改变。想法上的转变或许并不那么容易，这时候环境上的变化或许能带来一些刺激。这刺激是好是坏，我不得而知，但采取行动应该是必要的。

在新的一年，要告别那个软弱的自己，要变强，不只是外在能力的强大，更重要的是精神的强大。
